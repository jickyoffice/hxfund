# Qwen AI 功能快速使用指南

## 功能概述
Qwen AI 客户端是黄氏家族寻根平台的新功能，允许用户与通义千问AI模型进行对话，获取关于黄氏家族历史、族谱、字辈等相关信息的智能回答。

## 如何使用

### 1. 配置API密钥
1. 获取阿里云DashScope API密钥
   - 访问 https://dashscope.console.aliyun.com/
   - 登录并创建API密钥

2. 配置API密钥
   - 编辑项目根目录的 `.env` 文件
   - 将你的API密钥填入 `QWEN_API_KEY=` 后面

3. 重启后端服务以应用配置

### 2. 访问AI功能
1. 打开网站 `http://localhost:8080`
2. 点击导航栏中的 "Qwen AI" 选项
3. 在配置面板中选择模型和调整参数（可选）
4. 在输入框中输入你的问题
5. 点击"发送"按钮与AI对话

### 3. 多模态功能
1. 点击"上传图片"按钮
2. 选择一张图片（最大5MB）
3. 图片会在输入框上方预览
4. 输入相关问题
5. 点击"发送"按钮，AI将结合图片内容回答

## 支持的模型
- **qwen3.5-plus** (多模态) - 最新模型，支持文本和图像输入
- **qwen-max** - 复杂任务处理
- **qwen-plus** - 平衡性能模型
- **qwen-turbo** - 快速响应模型
- **qwen-mini** - 轻量高效模型

## 常见问题

Q: API调用失败怎么办？
A: 检查API密钥是否正确配置，网络连接是否正常

Q: 图片上传失败怎么办？
A: 检查图片格式是否支持（JPG、PNG等），文件大小是否超过5MB

Q: 如何获得更好的AI回答？
A: 选择适合的模型，调整temperature参数，提供更详细的问题描述

## 安全提示
